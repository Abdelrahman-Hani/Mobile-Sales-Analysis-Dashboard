📊 Mobile Sales Analytics Dashboard

This project presents an interactive Power BI dashboard built to analyze smartphone sales performance across multiple countries.

🔍 Key KPIs:

Total Revenue

Total Units Sold

Average Customer Rating

📈 Analysis Included:

Revenue by Country

Revenue by Color

Sales by Payment Method

Units Sold by Model

Units Sold by Storage

Monthly Revenue Trend

🎯 Project Goal:

To identify top-performing models, customer preferences, and revenue trends to support data-driven business decisions.

🛠 Tools Used:

Power BI

DAX

Data Modeling
